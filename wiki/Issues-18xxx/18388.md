Very minor point, but you could do the alignment:

```diff
-       return self.__class__(self.parent(),
-                             (self.i,
-                              matrix[0]*self.u + matrix[2]*self.v,
-                              matrix[1]*self.u + matrix[3]*self.v))
+       return type(self)(self.parent(),
+                         (self.i,
+                          matrix[0]*self.u + matrix[2]*self.v,
+                          matrix[1]*self.u + matrix[3]*self.v))
```
