<div id="comment:0"></div>

The default `copy(G)` of a graph `G` returns `G`, and then `relabel` can't do a relabeling inplace.

```
sage: G = Graph([[1, 2], [2, 3]], immutable=True)
sage: C = G.canonical_label() # Boom
```
