<div id="comment:4" align="right">comment:4</div>

Never use

```
try:
    ...
except:
    ...
```
(this catching thinks like `KeyboardInterrupt` and `SystemExit` which you really don't want).
You should always catch a *specific* exception, like:

```
try:
    ...
except AttributeError:
    ...
```
